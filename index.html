Why this exists

Based on Thariq Shihipar's "The unreasonable effectiveness of HTML" (Anthropic, May 2026). This repo packages the idea into a reusable Claude Code / Codex CLI / Gemini CLI skill (see install instructions), plus a frontmatter-less universal-prompt drop-in for everything else.

Want the same setup for your own reports? Fork this repo, enable GitHub Pages on the fork, and any .html file you push will go live at https://<your-username>.github.io/html-reports/<name>.html. Templates and skill come along for the ride.
