Implementation plan · v1

AI agent sandbox on Azure Container Apps

Stand up the MCP server, OE backend, and OE frontend on Azure Container Apps with VNet integration, custom domain, and Entra ID auth. Production blueprint validated on sandbox first.

Goals & non-goals

Goals. Three containers running with internal/external ingress wired up. PostgreSQL reachable from MCP via VNet peering. Custom domain with TLS. Entra ID enforced on frontend. Health endpoints return 200.

Non-goals. Production rollout, multi-region replication, GitHub Actions CI/CD pipeline. Those land in a separate plan.

Milestones
