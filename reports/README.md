# reports/

Drop your own generated `.html` reports here. They'll be served by GitHub Pages at:

```
https://andrewkobzev.github.io/html-reports/reports/<filename>.html
```

(After `git push`, allow ~30 seconds for the Pages build to complete.)

This folder is kept tracked in git so the URL path exists. Replace this README with your own reports when you have them — or keep it and add reports alongside.

## Privacy reminder

This repo is public. Anything you push under `reports/` is world-readable. For sensitive content see the "Privacy note" section in the main [README](../README.md).
